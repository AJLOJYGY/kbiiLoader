KBBI Loader - Interactive Indonesian Dictionary with Cartoon Adventure! 🎮
A fun, cartoon-themed web application for exploring Indonesian vocabulary (KBBI) with interactive characters, achievements, and smooth animations.

✨ Key Features:
🔍 Real-time word search with prefix matching

📖 Complete word collection with infinite scroll & filter

🎵 Background music player (upload your own kids.mp3)

🏆 Achievement system with 6 collectible badges

🦉 Interactive cartoon characters with speech bubbles

📊 Visitor statistics with localStorage

💡 Word of The Day generator

🎈 Floating balloons, stars & random click surprises

📱 Fully responsive design

🛠️ Built With:
HTML5, Tailwind CSS, Vanilla JavaScript

LocalStorage API, Web Audio API

CSS3 Animations (bounce, wobble, float, pop)
